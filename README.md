# Crypto-Terminal (Dashboard AI)

Дашборд для отслеживания криптовалют в реальном времени с AI.

![Скриншот](https://i.postimg.cc/wvjxknSW/cryptoterminal.png)

## Live Demo
[Посмотреть Демо](https://crypto-dashboard-blush-seven.vercel.app/)

## Возможности
- **Цены в реальном времени** - обновление каждые 60 секунд
- **Интерактивные графики** - исторические данные и аналитика
- **Список наблюдения** - избранные криптовалюты
- **Hugging Face AI** - вычисление рисков, тех.анализ, фунд. анализ и прогноз с помощью AI

## Технологии
- **Frontend**: React, Next.js, TypeScript, Chart.js
- **API**: CoinGecko API
- **Стили**: CSS Modules
- **State**: React Hooks
- **Деплой**: Vercel
